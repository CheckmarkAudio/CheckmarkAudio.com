# Project Documentation

This folder is the working memory for the Checkmark Audio website migration.

## Files

- `project-history.md` records major decisions and work completed.
- `context-history.md` captures useful background and operating context.
- `asset-system.md` defines the asset naming and folder system.
- `source-asset-inventory.md` records the current source/production asset audit.
- `production-readiness.md` tracks what is needed to leave Wix safely.
- `site-structure.md` defines the launch structure and future page split.
- `video-asset-capture.md` explains the remaining Wix video capture workflow.

Keep these files short, current, and practical. They are meant to help future work continue without re-discovering what already happened.
