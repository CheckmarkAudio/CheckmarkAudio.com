# Portfolio Page Requirements

The portfolio page is one of the most important trust pages.

## Must Include

- Embedded audio examples.
- Embedded videos.
- Checkmark Live Sessions.
- Before/after mix examples, if available.
- Artist photos.
- Music credits.
- Genre categories.
- Testimonials attached to real projects if possible.

## Portfolio Item Fields

- Artist name.
- Song/project name.
- Service performed.
- Engineer/producer credit.
- Audio or video embed.
- Short description.
