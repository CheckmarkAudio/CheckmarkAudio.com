# Source Playbook Summary

The source DOCX defines the website as a sales system, not a brochure. A first-time visitor should understand what Checkmark Audio does, who it is for, available services, starting prices, trust proof, booking/contact path, location, portfolio proof, and how the school connects to the studio.

## Required Pages From The Playbook

- Home.
- Services.
- Pricing.
- Book Now.
- Studios.
- Studio A.
- Studio B.
- Our Work / Portfolio.
- About Checkmark Audio.
- Team.
- Checkmark Audio School overview page or section.
- Contact.
- FAQ.
- Policies.

## High-Priority Competitive Features

- Clear pricing cards.
- Audio/video portfolio.
- Engineer-specific booking.
- Free consultation flow.
- Service-specific landing pages.
- Real client testimonials.
- Before/after mix examples.
- Studio comparison section.
- School pathway.
- Artist development pathway.
- Local Albuquerque mission statement.
- Strong mobile booking experience.

## Design Direction

- Mostly white/light mode.
- Black or dark gray text.
- Earthy accent colors.
- Warm neutral backgrounds.
- Clean cards.
- Rounded corners.
- Real studio photos.
- Consistent logo usage.
- Strong typography.
- Minimal clutter.
- Modern spacing.
- Clear buttons.
