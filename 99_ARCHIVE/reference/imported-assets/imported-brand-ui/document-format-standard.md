# Document Format Standard

## Markdown Front Matter

All major curriculum documents should begin with front matter:

```yaml
---
title:
program_section:
module:
chapter:
class_number:
audience: teacher | student | admin | public
status: draft | approved | archived
last_updated:
owner: Checkmark Audio School
---
```

## Standard Section Order

- Purpose
- Learning Objectives
- Required Materials
- Core Content
- Activities
- Assessment
- Teacher Notes
- Student Deliverables
