School Logo Proportion Drafts

Visual exploration only. The permanent Checkmark production/microphone mark stays untouched. These drafts test how the school book-and-headphones mark can share a more similar footprint, weight, and vertical distribution.

Current School MarkReference Starting Point

Current read: friendly and clear, but wider and lower. The book carries most of the visual mass, so it does not share the same tall centered footprint as the main brand mark.

Main Brand MarkPermanent

Target read: tall arch, strong side pads, centered vertical symbol, clean black mass, and a compact square footprint.

Option 1Tall Arch

Keeps the current book idea, but raises the arch and thickens the outer mass so it sits closer to the main mark’s square footprint.

Option 2Compact Book Core

Most like the main mark structurally: taller center, stronger black silhouette, and a small base gesture to echo the mic stand without becoming a mic logo.

Option 3Rounded Seal

Adds the lower curve from the permanent mark, making the school logo feel more like a sibling in the same family while staying book-first.

Option 4Vertical Pages

Boldest family match. The book becomes a heavier central symbol, closer to the microphone’s black mass, while page lines stay readable in reverse.
