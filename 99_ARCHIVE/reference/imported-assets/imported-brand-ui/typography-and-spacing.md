# Typography And Spacing

Typography should make the school feel practical, premium, and easy to work in for long sessions. Avoid oversized marketing type inside the app; students should scan, compare, and act.

## Type System

| Role | Size / Weight | Use |
| --- | --- | --- |
| Display | 40px / 700 | Rare module or marketing title |
| Page title | 30px / 700 | App page heading |
| Section title | 24px / 700 | Lesson or chapter section |
| Panel title | 18px / 600 | Card and widget heading |
| Body | 15px / 400 | Reading and instructional copy |
| Caption | 12px / 400 | Metadata and helper text |
| Label | 11px / 600 uppercase | Micro-labels only |
| Mono | 13px / 500 | Technical values and tabular numbers |

## Typography Rules

- Use Inter for all normal UI and reading text.
- Use JetBrains Mono only for technical values, progress percentages, and audio readouts.
- Use Title Case for module, chapter, and class titles.
- Use sentence case for descriptions, buttons, helper text, and error states.
- Avoid negative letter spacing except page-scale headings.
- Do not use viewport-scaled font sizes.

## Spacing

Use a `4px` base scale.

| Token | Value | Use |
| --- | --- | --- |
| `space-1` | 4px | Tight icon/text gaps |
| `space-2` | 8px | Compact control gaps |
| `space-3` | 12px | Form and row gaps |
| `space-4` | 16px | Card padding and grid gaps |
| `space-6` | 24px | Section spacing |
| `space-8` | 32px | Page-level spacing |
| `space-12` | 48px | Major document separation |

## Layout Rhythm

- App pages use `28-32px` content padding on desktop and `16px` on mobile.
- Cards use `16-18px` internal padding.
- Toolbars and headers stay compact: `44-60px` high.
- Dense operational views should use rows and tables before decorative card grids.

## Radius

- Controls: `8px`
- Inputs: `10-12px`
- Cards: `8px`
- Widgets: `12px`
- App shell: `16px`
- Pills: `999px`

## Print / Document Standards

- Prefer black text on white.
- Use gold for section anchors, key rules, and selected dividers.
- Avoid large filled backgrounds in printable materials.
- Tables should have clear horizontal rules and readable row spacing.
