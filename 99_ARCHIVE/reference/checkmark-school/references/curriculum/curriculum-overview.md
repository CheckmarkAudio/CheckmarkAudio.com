# Curriculum Overview

The Checkmark Audio School curriculum is a 12-month / 48-week studio-based program. Students build one major creative project across three modules: produce it, record it, then mix and master it.

Each module runs for 4 months / 16 weeks and contains 8 chapters. Each chapter runs for 2 weeks and includes 2 classes. Students attend one 4-hour class per week.

## Program Promise

Students learn by doing real audio work in a professional studio, with teacher guidance, critique, repetition, and clear deliverables.
