# Archive

Archived and ignored material lives here.

- `design-drafts/`: old website preview stacks and visual concepts that should not drive the future design by default.
- `ignored-local/` is ignored by Git and can hold HAR exports, temporary raw downloads, and other local-only files.
- `reference/`: old school, imported conversion, imported brand/UI, and legacy DOCX reference material.
- Do not use archived material as the active website source of truth unless it is deliberately promoted into `00_START/`, `01_WEBSITE/`, or `02_ASSETS/`.
